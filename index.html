How to set up a GitHub repo from the shell.

Without cloning the repo from GitHub.

  1. Create project folder on the computer
  2. Initialize git folder with $git init
  3. Create GitHub repo at github.com
  4. Link the git folder to the GitHub repo with: $git remote add origin git@github.com:user.name/github-repo-name.git
  5. Create, edit, stage and commit files
  6. Push to GitHub with: $git push origin main
